## ⚙️ Functionality Breakdown

### 🔗 Global CTA System
All buttons across the website are controlled using a **single global link variable**, making updates fast and scalable.

### ⏳ Timed Popup
A popup automatically appears after **10 seconds**, designed to capture user attention and improve conversions.

### 🎯 Conversion Layout
The page follows a structured flow:
- Hero Section (Hook + CTA)
- Features Section (Value)
- Pricing / Offer Section (Decision)
- Popup (Conversion Trigger)
